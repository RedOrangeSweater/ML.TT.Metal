# Диалект reu.sdlc_doc_intent (doc_intent)

Intent/effect для генерации документа: ссылки на KG ids, стиль/язык; без текстового content.

- **Префикс**: `reu` (rules-engine-universe); в tt-lang/tt-metal — `ttl`/`ttm`.
- **Операции**: `sdlc_doc_intent.claim` — параметры генерации (kg_ref, language, style).
- **Типы**: `!sdlc_doc_intent.claim_type`.
- **Реестр**: [../README.md](../README.md).
